# 有用的tips

### 0x00

CRYPTO_set_mem_functions 是 OpenSSL 中的一个函数，用于允许开发者自定义替换 OpenSSL 默认的内存分配、重新分配和释放函数

```c
int CRYPTO_set_mem_functions(
    void *(*m)(size_t),
    void *(*r)(void *, size_t),
    void (*f)(void *)
);
```
